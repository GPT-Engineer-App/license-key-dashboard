# license-key-dashboard

Create a dashboard to manage license keys for some software I'm selling.

I want to generate keys in batches of 100 of a time.

I want to mark when a key has been used.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/license-key-dashboard.git
cd license-key-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
